# 🥗 AI Diet & Workout Recommendation System

An AI-powered Diet and Workout Recommendation System developed using Machine Learning and Streamlit. The application predicts personalized diet and workout recommendations based on user health and lifestyle information.

---

## 📌 Project Overview

This project helps users receive personalized fitness recommendations by analyzing their health details such as age, gender, BMI, activity level, dietary preference, medical condition, and workout experience.

Users can compare recommendations generated using different Machine Learning algorithms.

---

## 🚀 Features

- Personalized Diet Recommendation
- Personalized Workout Recommendation
- BMI Calculation
- Health Status Prediction
- Daily Nutrition Summary
- Water Intake Recommendation
- Sleep Analysis
- Meal Plan Suggestion
- Health Tips
- Interactive Streamlit Dashboard
- Multiple Machine Learning Algorithms

---

## 🧠 Machine Learning Algorithms

- Random Forest
- K-Nearest Neighbors (KNN)

The application allows users to choose the algorithm and compare the predictions.

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Diet-Workout-Recommendation.git
```

Move into the project

```bash
cd AI-Diet-Workout-Recommendation
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
